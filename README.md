# Journal thingy

This is a minimal application designed to write a journal of everything you do. It supports attaching images, locations and logging your moods.

Here: https://journal-thingy.ingo.au

This was a project made for Horizons Crux.

## Running locally

```
bun install
cp .env.example .env
bun run db:push
bun run dev
```
